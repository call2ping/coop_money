# 💰 เปรียบเทียบดอกเบี้ย เงินฝาก / เงินกู้

เว็บแอปไฟล์เดียว (HTML/CSS/JS ล้วน ไม่มี dependency) สำหรับเปรียบเทียบดอกเบี้ย 2 อัตราในระยะเวลาที่ต้องการ

## ฟีเจอร์

- **โหมดเงินฝาก** — เทียบ 2 อัตรา แบบทบต้น (รายเดือน/ไตรมาส/6เดือน/ปี) หรือดอกเบี้ยคงที่
- **โหมดเงินกู้** — เทียบ **Effective Rate (ลดต้นลดดอก)** กับ **Flat Rate (คงที่)** พร้อมคำนวณอัตราที่แท้จริง (Effective APR) ของแบบ Flat
- กราฟเส้น + ตารางรายปี + สรุปส่วนต่างดอกเบี้ย
- คำนวณสดทันทีที่กรอกค่า

## ใช้งาน

เปิดไฟล์ [`index.html`](index.html) ในเบราว์เซอร์ได้เลย หรือดูออนไลน์ผ่าน GitHub Pages
